Welcome to CheckiO PyCharm Plugin

CheckiO PyCharm Plugin is plugin for PyCharm Edu to solve missions from CheckIO.

Features

Special CheckIO project in PyCharm Edu

Create CheckiO project in PyCharm Edu to start playing.

Available missions grouped by stations in Project view

All missions like in CheckiO are grouped by stations and you can see its solve and publication status.

Mission text panel with action toolbar

Mission's text and related actions are placed on Task Info tool window. You can open links from missions's text in internal browser.

User info panel

See you progress on user info panel.

Solution checking

To check you can run solution locally and then it's sent to CheckiO for testing.

Publications

For solved task you can see other's solution grouped by categories. Solution file is opened in local editor, so you can run it.

Hints

Like in CheckiO if you need help you can ask for it. Click on bulb and hint appears.
